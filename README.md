# cube-app

cube-app is a desktop app I built for the artist Alberto Frigo. His work is laid out as an 8 by 8 by 8 meter cube in the Alps, and every project in it has a code. You type a code into a grid and the app opens the matching file, whatever it is. Text and html show inline, and pdfs, audio, video and images open in a built in viewer.

![cube-app](assets/cube-app-gif.gif)

It is an Electron app. It remembers what you have looked at, runs fullscreen, and the interface is in English or Italian. Tested on Windows and Linux. There is a newer Rust version of it too.
